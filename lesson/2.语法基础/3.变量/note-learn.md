# 变量

## 什么是变量

变量是一块内存空间，用于保存数据

>计算机:CPU、内存、硬盘、输入输出设备
>内存:存取速度快，数据易丢失
>硬盘：存取速度慢，数据永久保存
>计算机程序的运行，仅与内存打交道

## 如何使用变量

1.声明(定义)变量

```js
var 变量名; //varable

```
**变量声明后，它的值为undefined**

2.给变量赋值

向变量的内存空间中存放数据


- 变量的值是可变的

变量可以被重新赋值，新的值会覆盖原来的值

-变量的名称
