# 变量和对象

原始类型：number、string、boolean、null、undefined

引用类型： object、function 函数

## 在变量中存放对象

1.通过变量读取对象的某个属性

```js
变量名.属性名
```
**当读取的属性不存在时，会得到undefined**

**当读取属性的对象不存在(undefined 或 null)时，程序报错**

2.通过变量读取对象中的某个属性

**当赋值的属性不存在时，会添加属性**

3.删除属性

```js
delete 变量名.属性名;
```

4.特殊的属性读取方式--属性表达式

给属性赋值，或读取属性时，可以使用下面的格式操作

```js
对象变量["属性名"]
```
-某些属性名中包含特殊字符

实际上，JS对属性名的命名要求并不严格，属性可以是任何形式的文字
